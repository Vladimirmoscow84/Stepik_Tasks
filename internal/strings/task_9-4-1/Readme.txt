У вас есть переменная message, которая содержит входные пользовательские данные.

Напишите код, который разбирает строку message на буквы и между каждой буквы проставляет пробелы.

Важно!

Если в строке message есть хотя бы одна буква в верхнем регистре (заглавная), тогда все буквы переводим в верхний регистр, иначе оставляем строку в текущем регистре.
Учитывайте то что ваш код должен работать как с кириллицей так и с латиницей.
Кириллица и латиница может встречаться в одной строке message
Новую строку записать в переменную result.

Sample Input 1:

TheNinaProject
Sample Output 1:

T H E N I N A P R O J E C T
Sample Input 2:

theninaproject
Sample Output 2:

t h e n i n a p r o j e c t
Sample Input 3:

ПриветМир
Sample Output 3:

П Р И В Е Т М И Р
